# 🚀 Contact Manager Application

A full-stack Contact Manager built using Java (HttpServer) and HTML, CSS, JavaScript.

---

## 🔹 Features

- Add Contact
- View Contacts
- Update Contact
- Delete Contact
- Search & Filter
- Sorting Table
- Dark Mode UI

---

## 🔹 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Java (HttpServer)
- Database: MySQL
- JSON Handling: org.json

---

## 🔹 Project Flow

User → Frontend → Backend API → Database → Response → UI

---

## 🔹 How to Run

### 1. Start Backend

Run Backend.java

Server runs at:
http://localhost:9090

---

### 2. Open Frontend

Open index.html using Live Server

---

## 🔹 API Endpoints

- /addContact
- /allContacts
- /deleteContact
- /updateContact

---

## 🔹 Notes

- Login is hardcoded (for demo)
- Basic backend (not Spring Boot)
- CORS enabled for frontend communication
